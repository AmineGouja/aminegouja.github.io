# Portfolio d'Amine Gouja

Portfolio personnel d'Amine Gouja, élève ingénieur en informatique à SUP Galilée,
conçu pour présenter son profil, ses compétences et une sélection de projets
académiques et techniques dans le cadre d'une recherche de stage.

## Profil

- Élève ingénieur en informatique
- Disponible pour un stage de 2 à 4 mois entre mai et août 2026
- Intérêt pour le développement logiciel, le web, l'algorithmique et les systèmes

## Objectif du site

Ce portfolio a été pensé comme une vitrine recruteur simple à parcourir :

- une page d'accueil synthétique avec profil, formation, compétences et contact
- un carrousel de projets permettant de parcourir rapidement les réalisations
- une page dédiée pour chaque projet avec contexte, stack, points techniques et mode d'exécution

L'objectif est de permettre à un recruteur ou à un tuteur de stage de repérer
rapidement :

- le niveau technique global
- la variété des technologies déjà manipulées
- la capacité à mener un projet de bout en bout
- le type de missions sur lesquelles Amine peut monter en compétence rapidement

## Projets mis en avant

Le portfolio présente actuellement plusieurs projets couvrant des sujets variés :

- API de similarité sémantique en Python / FastAPI
- optimisation combinatoire en Julia
- application web Java de gestion de maintenance
- compression de fichiers en C avec Huffman
- jeu Puissance 4 web en HTML / CSS / JavaScript / PHP
- simulation orientée objet en Java avec interface Swing

## Lancer le site en local

Le portfolio est un site statique. Il peut être ouvert directement dans un navigateur :

```bash
xdg-open index.html
```

Pour un rendu plus confortable pendant les modifications, il est recommandé
d'utiliser `Live Server` dans VS Code.

## Déploiement

Le site est prévu pour être publié simplement via GitHub Pages.
